jQuery Selecter

Jquery.selecter.js
jQuery plugin that lets you to create a beautiful customization for select element

Usage

1. Include jQuery and the plugin.
< script  src= "path/to/jquery.js" > < / script >
< script  src= "path/to/jquery.selecter.js" > < / script >
2. Include the plugin styles, either the compiled CSS...
< link  rel= "stylesheet"  href= "path/to/jquery-selecter.css" >
3. Initialize the plugin.
$(document).ready(function() {
  $('select').selecter();
});
4. Add multiple option
$(document).ready(function() {
  $('select').selecter({multiple: true});
});
5. Add custom classes
$(document).ready(function() {
  $('select').selecter({classname: 'dark'});
});
