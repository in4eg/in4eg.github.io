Практичні завдання:
1. Рекурсія
  1. Написати рекурсивну функцію для обчислення факторіалу
  2. Написати рекурсивну функцію для обчислення степені числа
  3. Написати рекурсивну функцію для обчислення суми цифр цілого числа (наприклад, сума цифр числа 3742 буде 3+7+4+2 = 16)
  4. Задачі з https://learn.javascript.ru/recursion#tasks
При виконанні цих завдань не можна використовувати цикли, рядки, масиви. Дозволена лише рекурсія і арифметика.
2. Виконати завдання 1.1 - 1.4 без використання рекурсії.
3. Визначити який варіант працює швидше (наприклад, скільки часу треба на 10000 обчислень)
