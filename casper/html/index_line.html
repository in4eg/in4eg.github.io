Casper Api для DApps

Casper создается, чтобы ваш DApp мог более эффективно и надежно хранить информацию ваших пользователей

Мы хотим видеть бурное развитие DApps, вне зависимости от блокчейн-платформы. Чтобы облегчить задачу по созданию собственного DApp, мы снимаем с разработчиков вопросы по хранению данных их пользователей, благодаря нашему Casper API. DApp на любой блокчейн-платформе со смарт-контрактом может использовать наш API для нужд хранения любого типа данных.

ПЕРЕДОВЫЕ ТЕХНОЛОГИИ CASPER:
CONVENIENCE
  • P2P для быстрой доставки файлов
  • Ethereum на proof of stake
  • Smart-Contract любого blokchain
SAFETY
  • Reliable coding of files
  • Files split into parts
Reliability
  • No single point of failur
  • Duplication of parts
  • Dynamic analysis and correction of failures

Цель

После отработки технологии хранения и стабилизации экосистемы Casper, мы переходим ко второй стадии проекта, формируя сеть децентрализованных быстродоступных распределенных виртуальных машин (используем HDD, CPU, RAM, GPU), для работы с запросами IoT, self-driving cars, AR/VR.

Все эти технологии требуют максимально оперативного отклика и подобная сеть сможет решить проблемы компаний из этих сфер бизнеса, что делает такое решение невероятно привлекательным для рынка.
